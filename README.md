# ReWear Admin Dashboard

Complete admin dashboard for ReWear sustainable fashion platform.

## Features

✅ Authentication (JWT)
✅ Dashboard Statistics
✅ Users Management
✅ Items Management  
✅ Orders Management
✅ Driver Verification
✅ Charity Management
✅ Deliveries Tracking
✅ Analytics & Reports

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment:
```bash
cp .env.example .env
# Update VITE_API_BASE_URL
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
